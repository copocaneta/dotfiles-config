Yabai and Skhd Config

This repository contains my standalone Yabai and Skhd configurations.

Keyboard Shortcuts

Changing Focus

⌥ Option + hjkl

Change focus between external displays (left and right)

⌥ Option + s

⌥ Option + g

Modifying the layout

Rotate layout clockwise

⇧ Shift + ⌥ Option + r

Flip along y-axis

⇧ Shift + ⌥ Option - y

Flip along x-axis

⇧ Shift + ⌥ Option - x

Toggle window float

⇧ Shift + ⌥ Option + t

Modifying Window Size

Maximize a window

⇧ Shift + ⌥ Option + m

Balance out tree of windows (resize to occupy same area)

⇧ Shift + ⌥ Option + e

Resize windows

⌃ Control + ⌘ Command + h

⌃ Control + ⌘ Command + j

⌃ Control + ⌘ Command + k

⌃ Control + ⌘ Command + l

Moving Windows Around

Swap windows

⇧ Shift + ⌥ Option + j

⇧ Shift + ⌥ Option + k

⇧ Shift + ⌥ Option + h

⇧ Shift + ⌥ Option + l

Move window and split

⌃ Control + ⌥ Option + j

⌃ Control + ⌥ Option + k

⌃ Control + ⌥ Option + h

⌃ Control + ⌥ Option + l

Move window to display left and right

⇧ Shift + ⌥ Option + s

⇧ Shift + ⌥ Option + g

Move window to prev and next space

⇧ Shift + ⌥ Option + p ⇧ Shift + ⌥ Option + n

Move window to space

⇧ Shift + ⌥ Option + 1

⇧ Shift + ⌥ Option + 2

⇧ Shift + ⌥ Option + 3

⇧ Shift + ⌥ Option + 4

⇧ Shift + ⌥ Option + 5

⇧ Shift + ⌥ Option + 6

⇧ Shift + ⌥ Option + 7

Starting/Stopping/Restarting Yabai

stop/start/restart yabai

⌃ Control + ⌥ Option + q

⌃ Control + ⌥ Option + s

⌃ Control + ⌥ Option + r
